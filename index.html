Computer Desktop

An interactive jQuery project by Becky Pollard

This fake desktop project was made with jQuery and jQuery UI mostly, with the addition of moment.js for the functionality of the clock in the toolbar. I made this in order to apply what I've been learning about JavaScrip and jQuery in a fun and creative way as part of HackerYou's Web Development Immersive 2018 bootcamp. It's not perfect, as it is a learning experience, but it was fun to make and a puzzle to complete. In the future I hope to dry out the JavaScript, improve performance, and add more applications to the desktop.
